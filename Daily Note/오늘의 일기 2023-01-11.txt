오늘의 일기
2023-01-11

아무것도 하기 싫다가 일단 밖으로 나와서 Java 공부를 시작했다.
과거에 대한 것들이 떠올라도 넘기면서 앞으로 전진해보자.
하는 마음가짐으로 Java 공부를 천천히 해나갔다.
결과는 성공적이었다.

그 이유는,
오늘 Java 책에서 가장 와닿았던 말이 있다.
"도전해 봤는데 어떤 문제로 인해 해결이 되지 않거나 시간이 20분 이상 지났다면
시간을 지체하지 말고 이번 시간에 살펴볼 온라인 도구를 이용해 일단은 실습을 진행합니다"

분명 최근 일주일간 나의 도전은 JS공부, Java 공부였고
그 이전에도 C 공부, Dart 공부, C++ 공부였다.

하지만 이 공부라는 행위에 있어서 나는 절대적인 문제가 있었다.
과거의 나. 혹은 공부를 하던 당시의 추억.
같은 과거에 대한 회상을 하느라 내가 해야하는 공부에 절대적으로 집중을 못하고 있었다.
그래서 계속 앞으로 나아가지 못했다.

나는 내가 수영 실력이 줄어들었다고 느낀 이유도, 같은 이유라고 생각한다.
그리고 이제는 코딩도, 수영도 성장해서 전진 할 수 있다고 생각한다.

왜냐하면, 오늘 Java를 공부하는 순간에도
수많은 생각이 들고, 수많은 과거가 보였다.
이걸 마주하거나 깊게 생각하지 않고, Java를 경험한다는 마음가짐으로
책을 천천히 음미하고, 천천히 코드를 적용해보았다.

그렇게 천천히 나아가다보니 나는 127 페이지까지 나아가있었다.
그리고, 새로운 지식과 발전을 받아들이기 싫어하는 관점이 바뀌기 시작했다.

훌륭한 4차산업혁명 시대에 훌륭한 기술을 배울 수 있는 기회가 넘치고,
누구보다 잘 할 수 있는 인재인데, 왜 활용하지 못했는가?
라는 질문이 던져졌고, 나는 최대한 익혀서 잘 활용해보기로 결심했다.
따라서, 계속 성장하면서 4차산업혁명을 주도하는 리더가 되고싶다.

개인적으로, 마냥 어린아이처럼 과거만 좇던 아이에서 성숙한 개발자가 되어가는 것 같다.

다른 무엇보다 코딩을 하고 빌드를 하는 순간을 계속 즐기고 싶다.

그리고 성장하는 사람으로서
1. 많은 정보와 그를 표현하는 것에는 책임이 따른다는 점을 꼭 인지하고 적용할 것이다.
2. 과거에 대한 수많은 경험과 정보를 굳이 쫓지 않는다.
(이들은 준비가 되면 하나의 경험치가 되고, 한 편의 아름다운 추억으로서 바라볼 수 있다고 믿는다.)
3. 현재의 나는 절대로 완벽하지 않다. 고로, 계속 전진하고 정진한다.
